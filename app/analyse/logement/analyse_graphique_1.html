Problématique

Ce graphique analyse l'évolution du prix moyen d'une maison dans les différents quartiers de Londres depuis 1995. L’objectif est de comprendre les tendances à long terme et de repérer les variations possibles liées à des événements économiques, politiques ou sociaux.

Origine des données

Les données proviennent du site officiel de la ville de Londres, data.london.gov.uk. Elles comprennent des informations mensuelles sur les prix moyens d'une maison par quartiers.

Traitement des données

  • Les données ont été extraites à partir de fichiers Exel mensuelles.
  • Les quartiers non pertinents ont été filtrés pour se concentrer sur les données spécifiques à chaque quartier de Londres.

Méthodologie statistique

  • Une visualisation en courbes (line plot) a été choisie pour représenter l'évolution du prix moyen d'une maison par quartier.
  • Les données ont été triées chronologiquement pour assurer une représentation précise des tendances.

Justification des choix

  • La visualisation en courbes permet de mettre en évidence les variations annuelles et les tendances à long terme.
  • L'utilisation de données mensuelles permet une analyse précise des évolutions sur une période de plus de deux décennies.

Utilisation du menu déroulant (Dropdown)

Le menu déroulant ci-dessus vous permet de sélectionner un quartier spécifique de Londres pour afficher les données correspondantes. Voici comment l'utiliser :

  • Sélectionnez un quartier : Cliquez sur le menu déroulant et choisissez un quartier dans la liste. Les données du graphique seront automatiquement mises à jour pour refléter les tendances du quartier sélectionné.
  • Comparaison entre quartiers : Vous pouvez changer de quartier à tout moment pour comparer les tendances entre différentes zones de Londres.
  • Focus sur des zones spécifiques : Cet outil vous permet de zoomer sur des quartiers particuliers pour une analyse plus détaillée.
