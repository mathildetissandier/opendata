Problématique

Ce graphique analyse l’évolution du nombre de logements vacants dans les différents quartiers de Londres de 2004 à 2022. Une carte est également disponible pour permettre une visualisation directe des quartiers avec le plus ou le moins de logements vacants. L’objectif est d’examiner les tendances à long terme et de repérer d’éventuelles fluctuations en lien avec des facteurs économiques, politiques ou sociaux.

Origine des données

Les données proviennent du site officiel de la ville de Londres, data.london.gov.uk. Elles comprennent des informations annuelles sur le nombre de logements vacants par quartier.

  • Un fichier GeoJSON des arrondissements de Londres a été utilisé pour créer des polygones géographiques.

    • Traitement des données

    • Les données ont été extraites à partir de fichiers Exel annuels.
    • Les colonnes pertinentes ont été sélectionnées, notamment le nombre de logements vacants et l'année.
    • Les zones non pertinentes ont été filtrés pour se concentrer sur les données spécifiques à chaque quartier.

    Méthodologie statistique

    • Une visualisation en courbes (line plot) a été choisie pour représenter l'évolution du nombre de véhicules par catégorie.
    • Une visualisation par carte a également été choisie pour mettre en évidence les différences d’évolution entre les quartiers.
    • Les données ont été triées chronologiquement pour assurer une représentation précise des tendances.

    Utilisation du menu déroulant (Dropdown)

    Le menu déroulant ci-dessus vous permet de sélectionner un quartier spécifique de Londres pour afficher les données correspondantes. Voici comment l'utiliser :

    • Sélectionnez un quartier : Cliquez sur le menu déroulant et choisissez un quartier dans la liste. Les données du graphique seront automatiquement mises à jour pour refléter les tendances du quartier sélectionné.
    • Comparaison entre quartiers : Vous pouvez changer de quartier à tout moment pour comparer les tendances entre différentes zones de Londres.
    • Focus sur des zones spécifiques : Cet outil vous permet de zoomer sur des quartiers particuliers pour une analyse plus détaillée.

    Utilisation du Slider

    Le Slider vous permet de sélectionner une année spécifique entre 1997 et 2023 pour afficher les données correspondantes. Voici comment l'utiliser :

    • Sélectionnez une année : Faites glisser le bouton sur le curseur pour sélectionner la date souhaitée. Les données du graphique seront automatiquement mises à jour pour refléter les tendances du quartier sélectionné.
    • Comparaison entre années : Vous pouvez changer d'année à tout moment pour comparer les tendances entre différentes zones de Londres.
